## Requirements

| Requirement | Minimum |
|-------------|---------|
| Docker Engine | 20.10+ |
| OS | Linux |
| CPU | 4 vCPU |
| Memory | 8GB RAM |
| Storage | 50GB |
| Domain | Optional (required for HTTPS) |

## Services

| Service | Purpose |
|---------|---------|
| **app** | Next.js frontend and API |
| **worker** | Test execution (Playwright/K6) |
| **postgres** | Database |
| **redis** | Job queues |
| **minio** | Artifact storage (S3-compatible) |
| **traefik** | HTTPS reverse proxy (production only) |
